Firebase Configuration Required

This application requires Firebase configuration to run properly.

  1. Go to your Lovable project dashboard
  2. Click on Project Settings in the sidebar
  3. Navigate to the Environment Variables section
  4. Add these required variables:
    • VITE_FIREBASE_API_KEY
    • VITE_FIREBASE_PROJECT_ID
    • VITE_FIREBASE_AUTH_DOMAIN
    • VITE_FIREBASE_APP_ID
  5. Click Save and then Publish your app

For more information, see the environment variables documentation.
