Firebase Configuration Error

This site requires Firebase configuration to function properly.

If you are the site administrator, please add the following environment variables to your hosting platform:

  • VITE_FIREBASE_API_KEY
  • VITE_FIREBASE_PROJECT_ID

How to Fix This Issue:

  1. Go to your Vercel Dashboard
  2. Select your Ice Guardian project
  3. Click on "Settings" in the top navigation
  4. Select "Environment Variables" from the left menu
  5. Add these required variables:
    • VITE_FIREBASE_API_KEY - Your Firebase API key
    • VITE_FIREBASE_PROJECT_ID - Your Firebase project ID
  6. Click "Save" and redeploy your application

Troubleshooting Steps:

  1. Check if your Vercel environment variables are set correctly
  2. Clear your browser cache and refresh
  3. Try using an incognito/private browser window
  4. Check browser console for specific errors (F12 or right-click → Inspect → Console)
